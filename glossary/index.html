Glossary

Bell Labs

Bell Labs is the research divsion of AT&T.

C

C is a popular high-level programming language. It is commonly used in environments where speed is key and control over many aspects of the program is needed. C is used in embedded systems and also in operating system design, most notable of which is the UNIX kernel.

Command line interface

CLI

The command line interface, shortened to CLI, is a user interface that primarily uses the concept of commands typed on a screen to accomplish tasks. While CLIs can be non-intuitive, they can help massively with quick modification and complicated tasks that would otherwise be prohibitively difficult with a GUI that can be done with only a few commands with a CLI.

Daemon

In UNIX and UNIX-like systems, a daemon is a program that performs its duties in the background. They do not usually have GUIs, and usually provide interfaces from the hardware, firmware and operating system to user-space software.

The Windows equivalent is called a service.

Graphical user interface

GUI

A GUI is a user interface that heavily utilises GPU-drawn graphics such as bitmapped images, shapes, and real-life metaphors such as icons in contrast of a command-line interface which uses only rows and columns of text drawn uniformly to a screen.

GNU Project

The GNU Project was founded by free software activist Richard Stallman as a community effort to create a fully free and open-source operating system based upon UNIX. The GNU Project uses the GPL, a license which allows the end-user unlimited privileges to copy, modify and redistribute the software, and for any modifications to be licensed as openly. Currently, the most popular implementation of GNU tools is in GNU/Linux, with the Linux kernel being seperate from the project but very much dependent on the tools provided by it.

High-level programming language

High-level programming involves many structures and syntax that help the programmer to avoid dealing with the low-level aspects of the hardware and operating system.

Examples of high-level programming languages include C++, Java, Haskell and Python.

Massachusetts Institute of Technology

MIT

MIT is an institute for higher learning that specialises in technological progress. MIT alumni have helped shape the worlds of technology and computer science.

Minicomputer

Monolithic

A monolithic application is one that performs all of the processes needed to complete a specific task and is self-contained (that is, doesn't use any other programs to carry out its function). Parts of a monolitic application cannot be used to complete other tasks.

MULTICS

MULTICS, short for MULTiplexed Information and Computing Services, was an early collaborative effort by Bell Labs, General Electric and MIT to develop an efficient time-sharing system that could run on large mainframes. The project was later abandoned, and some engineers from Bell Labs who previously worked on Multics moved on to develop UNIX.

Multi-user

A multi-user operating system is one that supports use by more than one computer user at one time. This is often acheived by remote login and/or multiple terminals. Time-sharing systems are multi-user systems.

Operating system

An operating system is a collection of software that provides an interface between user programs and the low-level hardware and firmware. The operating system handles a variety of tasks, including startup, shutdown, I/O, file system read/write, etc.

Piping

Piping is a feature in command-line interfaces that allows the output from one program to be interpreted as the input to another program. It is called "piping" because the character that acheives this for many interfaces, starting with UNIX, is the vertical bar |, sometimes called a "pipe".

Regular expression

A regular expression is a string of text interpreted as a pattern. It uses metacharacters such as ^ and $ to represent places in the text and constructs such as [A-Z], ? and + to define the possible text strings that can fit the pattern.

Their syntax can be confusing at first, but their capabilities, combined with their wide adoption and use in CLI programs, makes them invalulable to programmers and power users.

Time-sharing

Time-sharing, when relating to computing, is where one computer hosts multiple users running different tasks at the same time. Time-sharing was utilised to take advantage of the free computer resources available when one particular user is not processing a task (e.g. when they are entering a command or away from the computer).

Wildcard

The wildcard character (*) is used in many computer programs to represent a place where any string of text could reside, even the zero-length string (e.g. *.html could mean ".html", "a.html", "b.html", "ab.html", "bcc.html", etc.). Wildcard characters are a basic way to specify multiple strings of text through a string specification (see regular expressions for a more advanced method).

User space

User space is used to refer to programs run by the user of the computer voluntarily, as opposed to kernel space, where basic programs such as drivers and interfaces run and interact with each other. It has been said by many, including Linus Torvalds (Linux kernel creator/maintainer) that it is imperative to keep these two categories of programs seperate.
